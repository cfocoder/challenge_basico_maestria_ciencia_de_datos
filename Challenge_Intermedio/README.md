# Challenge Intermedio: Wine Quality Prediction

## Descripción

Este challenge intermedio se enfoca en el uso de PySpark y Spark ML para implementar un modelo predictivo de clasificación en el Wine Quality Dataset de UCI Machine Learning Repository.

### Objetivos
- Aplicar algoritmos de machine learning en grandes bases de datos utilizando Spark ML.
- Descargar, limpiar y preparar datos.
- Entrenar un modelo de regresión logística para predecir la calidad del vino.
- Evaluar el modelo y documentar resultados.

### Evaluación
- Limpieza y preparación de datos: 20%
- Selección del modelo: 20%
- Exactitud y evaluación: 30%
- Documentación: 30%

## Contenido del Notebook

El notebook `reto_intermedio_wine_quality.ipynb` contiene la implementación completa del challenge:

- **Configuración de Spark:** Inicialización de sesión Spark en Databricks.
- **Carga de Datos:** Descarga y carga del Wine Quality Dataset.
- **Exploración de Datos:** Análisis exploratorio, estadísticas descriptivas y visualizaciones.
- **Preprocesamiento:** Limpieza, transformación y preparación de features.
- **Modelado:** Entrenamiento de modelo de regresión logística con Spark ML.
- **Evaluación:** Métricas de rendimiento, matrices de confusión, curvas ROC.
- **Conclusiones:** Interpretación de resultados y recomendaciones.

El notebook está diseñado para ejecutarse en Databricks, ya sea directamente o conectado desde VS Code.
